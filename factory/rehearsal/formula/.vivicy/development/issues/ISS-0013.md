## Scope

Implement the public surface. The package exposes a single public entry function that evaluates a formula string against a sheet, plus a factory that creates an empty sheet, the error-inspection predicate, and the error-code reader that returns an error value's code string (both from ISS-0001). No other module is part of the public surface; callers import only from the package entry point.

Implement `evaluate(formula, sheet)`: it accepts a formula string and a sheet, runs the formula through the Tokenizer (ISS-0004), Parser (ISS-0005), and Evaluator (ISS-0011), and returns the resulting value. A leading equals sign on the formula is optional and stripped, and any lexical, parse, evaluation, or reference failure is returned as one of the typed error values rather than thrown.

Implement sheet construction. The factory creates an empty sheet (over the Sheet of ISS-0003) and exposes setting a cell's literal value or formula by A1 address and reading a cell's current value; setting a formula cell registers its dependencies (via the DepGraph/recalculation of ISS-0012) so a later read reflects recalculation against the cells it references. The public functions are pure with respect to inputs other than the sheet they are given: evaluating the same formula against the same sheet returns the same value, and `evaluate` never mutates the sheet it reads. Errors surface only as typed error values, so a caller can branch on them with the error-inspection predicate.

Enforce the architecture boundaries the canonical baseline defines: Formula is a small, dependency-free JavaScript (ESM) library that runs entirely in memory and talks to no external service, database, file system, or network; the Sheet is the single source of truth for stored cell values and no other module mutates a stored value directly; the Tokenizer, Parser, Evaluator, and function families are pure transformations holding no spreadsheet state; and a formula string enters through the API, is lexed, parsed, walked by the Evaluator (which resolves references against the Sheet and calls registered functions), and returns a single value to the API.

## Out Of Scope

The internal modules' own behavior (owned by ISS-0001 through ISS-0012, composed here). Adding any new function, operator, or error code. Any external I/O, persistence, or network — explicitly excluded by the in-memory architecture this issue enforces. This issue assembles and exposes the public entry point and the empty-sheet factory; it does not reimplement the pipeline stages it wires together.
